11 Materi Web untuk Guru

Klik kartu untuk detail + sample kode HTML siap pakai

Studi Kasus: Web dengan HTML+CSS+JS vanilla. Fitur: profil 7+ tahun, jadwal, filter materi.

portfolio-guru.html
<section class="max-w-4xl mx-auto text-center">\n  <h1 class="text-4xl font-semibold">Siti Nurhaliza, S.Pd</h1>\n  <p>Guru Bahasa Indonesia | SMP Negeri 1</p>\n</section>

Referensi: Struktur halaman terpisah meliputi: kelas-7a.html, tugas.html, dan UTS online yang datanya ditarik langsung dari sel Google Sheets.

Inspirasi: Menggunakan JavaScript Vanilla + API Fetch ke Google Sheets serta memanfaatkan localStorage browser untuk merekam papan skor kuis murid.

Marketplace Guru & Hub: Memetakan direktori nama pengajar, filter kategori rumpun mata pelajaran, hingga tombol integrasi langsung menuju pesan instan WhatsApp guru piket.

Inspirasi: Manajemen penyimpanan dokumen portofolio kreasi orisinal siswa per angkatan untuk meningkatkan transparansi mutu akademis sekolah.

Arsitektur Ringan: Rekam video pembelajaran harian, upload sebagai status Unlisted di YouTube, lalu panggil ID-nya lewat kode website agar streaming super cepat, gratis, dan bebas kuota hosting.

AI-Powered Engine: Guru menginput parameter mata pelajaran, tingkatan kelas, dan capaian semester → Sistem mengirim prompt ke Gemini API untuk menyusun draf RPP 1 lembar tanpa ketik dari nol.

Mulai Pendampingan Perencanaan →

Sistem LKPD Pintar: Guru memasukkan Kompetensi Dasar (KD). AI otomatis menyusun soal studi kasus pemecahan masalah beserta rangkuman lembar materi penunjangnya.

Bank Soal Otomatis: Generator otomatis merancang 10-20 variasi soal pilihan ganda lengkap dengan rubrik penilaiannya, tersinkronisasi rapi ke basis data internal.

Konsep Mandiri: Penanaman konteks kurikulum spesifik via System Prompt. Murid bisa berkonsultasi seputar rumus fisika, materi sejarah, atau kisi-kisi ulasan materi kapan saja secara mandiri.

Fitur Laboratorium Bahasa Digital: Integrasi video percakapan native speaker, modul tes kosa kata interaktif, hingga evaluasi struktur kalimat berbasis aplikasi web Jamstack.
